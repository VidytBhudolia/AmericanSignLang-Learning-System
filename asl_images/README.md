# ASL Alphabet Images

## Instructions

Place ASL alphabet images in this folder with the following naming convention:

- `A.png` or `A.jpg` - Image for letter A
- `B.png` or `B.jpg` - Image for letter B
- ... and so on for all 26 letters (A-Z)

## Image Requirements

- **Format**: PNG, JPG, or JPEG
- **Naming**: Single letter followed by extension (case-insensitive)
  - ✅ `A.png`, `B.jpg`, `C.PNG`, `D.JPG`
  - ❌ `letter_a.png`, `alphabet_b.jpg`

## Where to Get ASL Images

You can:

1. **Download from the web**: Search for "ASL alphabet chart" or "ASL letter [A-Z]"
2. **Create your own**: Take photos of ASL hand signs
3. **Use existing datasets**: Look for ASL alphabet image datasets

## Recommended Sources

- ASL University: https://www.lifeprint.com/asl101/fingerspelling/
- ASL alphabet charts on Google Images (ensure proper licensing)
- Create screenshots from ASL learning videos

## Example Structure

```
asl_images/
├── A.png
├── B.png
├── C.png
├── D.png
├── E.png
... (continue for all 26 letters)
├── X.png
├── Y.png
└── Z.png
```

## Notes

- The app will work without images, but they greatly enhance the learning experience
- Images help learners see the correct hand position for each letter
- Higher resolution images (at least 300x300px) work best
